Here we the commands that we will be going to use to operate the typescript.
  1. Create package.json
      npm init -y
  2. Check Globally installed packages
      npm list -g
  3. Install Typescript
      npm i typescript -g
  4. Install tsc (transpiler)
      npm i tsc -D
  5. To Run Typescript File
      npm run tsc<filename> <extension>
  1. Create tsConfig file (tsconfig.json)
      tsc --init
    • compilerOptions { target: JS version eg: es2017,ES6}
